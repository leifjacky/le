运行python程序

直接使用python命令行

$python    
>>>print(‘Hello World!’);            
Hello world!

写一个.py结尾的python程序

$echo "print('Hello World!')" >> hello.py
$python hello.py
Hello World!

使用可执行脚本的方式运行程序

编辑helloScript.py，输入以下代码

#!/usr/bin/python
print('Hello World!')

修改helloScript.py权限

chmod 755 helloScript.py

运行程序

./helloScript.py
Hello World!



基本数据类型

常用数据类型

变量 数据类型
a = 10 int 整形
a = 1.3 float 单精度浮点数
a = True 布尔型（True / False）
a = ‘Hello!’ 字符串

查看数据类型

>>>a = 1.3
>>>print a,type(a)
1.3 <type 'float'>

sequence 序列

tuple （定值表、元组）

>>>s1 = (2, 1.3, 'love', 5.6, 9, 12, False)   # s1是一个tuple

list （表）

>>>s2 = [True, 5, 'smile']    # s2是一个list

tuple 和 list 的区别

一旦建立，tuple的各个元素不可再变更，而list的各个元素可以再变更。

元素引用方式

>>>print s1[0]              # 下标0
>>>print s3[1][2]           # 二维行下标1，列下标2
>>>print s1[:5]             # 从开始到下标4 （下标5的元素 不包括在内）
>>>print s1[2:]             # 从下标2到最后
>>>print s1[0:5:2]          # 从下标0到下标4 (下标5不包括在内)，每隔2取一个元素 （下标为0，2，4的元素）
>>>print s1[2:0:-1]         # 从下标2到下标1
>>>print s1[-1]             # 序列最后一个元素
>>>print s1[-3]             # 序列倒数第三个元素

以上引用方式同样适用于字符串，字符串是特殊的元组。

选择语句

if 语句

i = 1
x = 1
if i > 0:
    x = x + 1
print x

程序输出 2

缩进

python 中没有括号{}，在 if 语句中如果 if 后面跟了多个语句，则通过缩进的方式确定当前选择结构下的语句内容，具体看例子：

 i = 1
 x = 0
 y = 0
 if i > 2:
    x = 1
    y = 2
print x, y

程序输出 0 0，说明 x = 1 和 y = 2 两句确实包含在 if 语句中。

多重选择语句

i = 1

if i > 0:
    print 'positive i'
    i = i + 1
elif i == 0:
    print 'i is 0'
    i = i * 10
else:
    print 'negative i'
    i = i - 1

print 'new i:',i

这里有三个块，分别属于if，elif，else引领。同时 if 中也可以嵌套 if。

i  = 5
if i > 1:
    print 'i bigger than 1'
    print 'good'
    if i > 2:
        print 'i bigger than 2'
        print 'even better'



循环语句

前面在 if 中说到了缩进机制，缩进机制同样适用于循环语句。

for 循环

在讲 for 循环前先说一个函数 range(x)，返回 [0,1,2,3,4] 序列。

基本格式

for 元素 in 序列: 
    statement

具体用法

for a in [3,4.4,'life']:
    print a
for a in range(10):
    print a**2

while 循环

基本格式

while 条件:
    statement

具体用法

i = 0
while i < 10:
    print i
    i = i + 1

中断循环

continue, break 与 C 用法类似

for i in range(10):
    if i == 2: 
        continue
    if i == 3:
        break;
    print i



函数

函数定义

def square_sum(a,b):
    c = a**2 + b**2
    return c
  • return可以返回多个值，以逗号分隔。相当于返回一个 tuple（定值表）。return a,b,c 相当于 return (a,b,c)
  • return并不是必须的，当没有return, 或者return后面没有返回值时，函数将自动返回None。None是Python中的一个特别的数据类型，用来表示什么都没有，相当于C中的NULL。None多用于关键字参数传递的默认值。

参数传递

a = 1

def change_integer(a):
    a = a + 1
    return a

print change_integer(a)      #注意观察结果
print a      #注意观察结果

b = [1,2,3]

def change_list(b):
    b[0] = b[0] + 1
    return b

print change_list(b)      #注意观察结果
print b      #注意观察结果

  • 第一个例子，我们将一个整数变量传递给函数，函数对它进行操作，但原整数变量a不发生变化。

  • 第二个例子，我们将一个表传递给函数，函数进行操作，原来的表b发生变化。
